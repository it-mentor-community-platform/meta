# Бизнес аналитика - профили

Профиль участника сообщества включает в себя:
- Имя, фамилия, ссылка на профили в Telegram, Github
- Роль
- Достижения (achievements)
- Проекты и ревью

## Достижения (achievements)

Элемент геймификации. Аналогия - ачивки в играх.

Пользователь может получать ачивки, выполняя определённые действия - сдавать проекты, делать ревью. Полученные ачивки пользователь может по желанию отобразить в профиле.

### Виды ачивок

За проекты:
- "Первый проект" - за сдачу первого (любого, не обязательно виселицы) проекта данным пользователем
- "Все проекты" - за сдачу всех проектов с 1 по 7 в любом порядке и на любых языках
- "Билингв" - за сдачу проектов на 2 языках
- "Полиглот" - за сдачу проектов на 4+ языках
- "Спринтер" - сданы проекты с 1 по 6. Разница между таймстампами сдачи первого и шестого проектов - меньше чем 6 месяцев
